# Prep Area

Dice Masters collection tracker and team builder. 

## What it does
- Browse the full Dice Masters card database offline.
- Search and filter by set, energy type, rarity, alignment, format bans, and more.
- Track collection counts for cards, dice, foils, and want lists; data is stored locally.
- Build teams, manage dice counts, export/import data, and compare trades.
- Utilities: proxy dice list, life counter, glossary, keywords, and rulings references.
- Tade tools, for creating list of both spare cards/dice and cards/dice needed.
